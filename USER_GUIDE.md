## 1. このアプリでできること

- コモンマーモセット／マウス脳の **空間メタボロミクス（MSI）** と **組織染色画像（HE・IF）** を同じ空間で重ね合わせて観察できます。
- あらかじめ登録された **ROI（関心領域）** に加え、画面上で **手描きの ROI** を追加登録できます。
- 選んだ ROI 内での **化合物の平均強度** をその場で棒グラフ比較できます。
- 元データと手描き ROI を含む **ZIP バンドル** をワンクリックでダウンロードできます。

## 2. はじめに：動作環境と起動

### 2-1. 推奨ブラウザ

- Google Chrome / Microsoft Edge / Mozilla Firefox / Safari のいずれも最新版
- 画面幅は **1024px 以上** を推奨（モバイル表示は未最適化）

### 2-2. 公開サイトを開く

ブラウザで次の URL を開きます。

```
https://ryoma67744.github.io/marmoset_mouse_atlas/
```

ルート URL にアクセスすると、自動的にサンプルデータセット（`cor_slide_1_10`）が読み込まれた Viewer に遷移します。

### 2-3. 他のデータセットに切り替える

URL 末尾のクエリパラメータ `?atlas=<フォルダ名>` を変えると、データセットを切り替えられます。

| URL 例 | 内容 |
| --- | --- |
| `…/viewer/index.html?atlas=cor_slide_1_10` | HE TIFF + MSI 4 化合物のコロナルスライス |
| `…/viewer/index.html?atlas=test1` | HE / IF / Mam-Sagittal を含むサンプル |

### 2-4. バージョン表示

画面右下に小さく **`v:xxxxx`** というビルドタグが表示されています。最新修正が反映されていないように見えるときは、このタグを確認したうえで `Ctrl+F5`（Windows）または `Cmd+Shift+R`（macOS）で強制再読み込みしてください。

## 3. 画面の見かた

起動すると、画面は大きく 3 列に分かれています。

```
┌──── ヘッダー（タイトル／Download ボタン） ───────────────────────┐
│                                                                 │
│  SPECIMEN   │       Image Viewer          │   ANALYSIS          │
│  ROI LIST   │  （選択画像の合成表示）      │  （棒グラフ）        │
│             ├─────────────────────────────┼─────────────────────┤
│             │  IMAGES（サムネイル行）      │  Mass 測定情報      │
│                                                          v:xxxxx
└──────────────────────────────────────────────────────────────────┘
```

| 領域 | 役割 |
| --- | --- |
| **ヘッダー** | タイトル表示、`Download` ボタン（ZIP 出力） |
| **SPECIMEN** | 検体メタ情報（ID, Organism, Strain, Age, Sex） |
| **ROI LIST** | 解析対象 ROI の一覧・表示切替・手描き登録 |
| **Image Viewer** | 複数画像を重ね合わせたメインキャンバス |
| **IMAGES** | 各ソース画像のサムネイル（クリックで表示切替） |
| **ANALYSIS** | 選択中の ROI における化合物の平均強度を比較する棒グラフ |
| **Mass 測定情報** | 今後の機能拡張用エリア（現状はプレースホルダ） |

## 4. 基本ワークフロー

最初に試していただきたい流れです。

1. **公開 URL を開く** → サンプルの HE 画像 + MSI が重ね合わせで表示されます。
2. 右上の **Select Images** をクリックし、表示したい画像（HE・MSI など）にチェックを付ける。
3. 左の **ROI LIST** から気になる ROI をクリック → Image Viewer 上にハイライト、ANALYSIS グラフが更新されます。
4. 必要なら ANALYSIS 上部のプルダウンで比較する化合物を選び直します。
5. 画面上で独自の領域を解析したいときは `+新規` で **手描き ROI** を追加します。
6. 最後に右上の **Download** で、手描き ROI を含めた ZIP バンドルを保存します。

## 5. 画像の重ね合わせと表示調整

### 5-1. 画像の選択

- 中央上部の **`Select Images`** ボタンを押し、ドロップダウンのチェックボックスで表示したい画像を選びます。
- 下段 **IMAGES** のサムネイルを **クリック** しても同じ効果です。選択中のサムネには青枠が付きます。
- ボタン表記は現在選択中の画像名の連結に切り替わります（例：`HE Stain + MSI (DA) + MSI (NE)`）。

### 5-2. 重ね合わせのルール

- **HE / IF** はグレースケール正規化で描画されます。
- **MSI** は加算合成（`lighter` ブレンド）で描画されるため、複数化合物を同時選択すると色が混ざって見えます。
- 化合物ごとの色は固定です：

  | 化合物 | 色 |
  | --- | --- |
  | DA | 赤 |
  | 5-HT | 緑 |
  | NE | 黄 |
  | Ach | 橙 |
  | GABA | シアン |
  | Glu | マゼンタ |
  | TIC | 無彩色（白） |

### 5-3. コントラスト調整（MSI）

1. IMAGES 内の MSI サムネイルに **マウスを乗せる** と、右上に歯車アイコンが現れます。
2. クリックすると `vmin` / `vmax` の数値入力が開きます。
3. 値を変更すると、Image Viewer とサムネイルの両方に **即時反映** されます。

> 真っ黒に見えるときは `vmax` が高すぎる可能性があります。一度既定値に戻すかサムネに合わせて調整してください。

## 6. ROI の表示・選択・手描き追加

### 6-1. 既存 ROI の操作

| 操作 | 結果 |
| --- | --- |
| ROI 名をクリック | その ROI をアクティブ化し、ANALYSIS を更新 |
| ROI 行のチェックボックス | その ROI 単体の表示／非表示 |
| ヘッダの **`Show`** トグル | 全 ROI の一括表示／非表示 |

ROI の色はデータセット側で定義されており、手描きで追加した ROI には衝突しないパレット色が自動で割り当てられます。

### 6-2. ROI を手描きで追加する

1. ROI LIST 上部の **`+新規`** をクリックします。ボタンが `中止` に変わり、Image Viewer のカーソルが十字になります。
2. Image Viewer 内を順番にクリックして **頂点を 3 点以上** 追加します。
3. 確定は次のいずれかで行います。
   - 最初の頂点をもう一度クリック
   - ダブルクリック
   - **Enter キー**
4. 名前の入力ダイアログが出るので、任意の名称を入力します。
5. 取り消したいときは **Escape キー** または `中止` ボタンを押してください。

追加した ROI は `User ROI 1, 2, …` として ROI LIST に登録され、Download で出力する ZIP の `.xlsx` / `.json` にも列／フィールドとして書き出されます。

> ROI 座標は MSI ピクセル空間（生の列・行インデックス）で記録されるため、画面上の回転や拡大縮小には影響されません。

## 7. ANALYSIS（棒グラフ）で比較する

- ANALYSIS パネル上部の 3 つのプルダウンで、比較する化合物を最大 3 つまで選べます。`-` を選ぶとその系列は表示されません。
- ROI LIST から ROI をクリックするたびにグラフが更新されます。
- 各バーは **その ROI 内の平均強度（Mean Intensity）** を示します。
- 棒の色は MSI の化合物カラー（5-2 表）に対応します。

---

## 8. 解析結果のダウンロード（ZIP バンドル）

ヘッダー右上の **`Download`** ボタンで、以下をひとまとめにした ZIP を生成します。

| ファイル | 内容 |
| --- | --- |
| `atlas.json` | ROI 定義・world 座標・画像参照などのメタファイル |
| `data/*.xlsx` | 元 MSI データに **手描き ROI 列を追加** した版 |
| `data/*.TIF` | ソースの HE/IF TIFF（存在する場合） |
| `data/*.json` | ソース付属のメタ情報（存在する場合） |

手描き ROI を含めた **再現可能な配布物** として、そのまま共有・保管できます。

> 大きな TIFF / xlsx を同梱する場合は、生成に数十秒かかることがあります。進捗表示は出ませんが、ブラウザの通信インジケータで処理中かを確認できます。
